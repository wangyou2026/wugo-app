# 忘忧围棋 App

## 项目介绍
忘忧围棋是一个专业的单机围棋做题App，拥有海量题库（15万+题目），完全免费，无需联网。

## 核心功能
### 1. 超大题库系统
- 15万+题目，涵盖死活、手筋、官子、定式、中盘战斗
- 完全离线，题库预装，可自由扩展
- SGF标准格式，兼容现有题库资源

### 2. 智能分级系统
- 双轨难度体系：传统段位制 + 星级制
- 动态难度调节：根据用户水平自动推荐题目
- 弱点诊断：AI分析错题类型，生成专项训练

### 3. 多样做题模式
- 每日一练：根据水平每日推送10-20题
- 闯关模式：关卡制，逐步解锁，成就感强
- 专项训练：针对特定知识点集中突破
- 错题重练：自动收录错题，间隔重复算法
- 计时挑战：限时解题，培养读棋速度
- 自由浏览：按分类/难度/来源随意选题

### 4. 本地AI引擎
- 基于KataGo轻量分支，离线运行
- 一题多解：显示AI认为的所有合理下法及胜率
- 变化图分析：自动生成完整变化树
- 难度评估：AI自动评估题目真实难度
- 生成变形题：基于经典题目自动生成变体

### 5. 拍照识题功能
- 手机摄像头拍摄实体书上的死活题
- 本地OCR + 棋盘识别，无需联网
- 自动转换为可交互的做题界面
- 支持识别后保存到"我的收藏"

## GitHub Actions 云编译

### 自动编译配置
项目配置了GitHub Actions自动编译流程，可以实现：

1. **自动编译**：每次代码推送自动编译
2. **云端编译**：无需本地Android环境
3. **APK生成**：自动生成Debug和Release APK
4. **持续集成**：保持项目编译状态

### 使用方法
1. 在GitHub创建仓库并上传项目
2. GitHub会自动检测 `.github/workflows/android.yml` 配置
3. 每次推送代码时自动编译
4. 从Actions页面下载生成的APK文件

### APK文件下载
编译完成后，可以从GitHub Actions页面下载：
- Debug APK：包含调试信息的测试版本
- Release APK：优化后的发布版本

## 技术架构
### Android平台
- **开发框架**：Jetpack Compose（现代UI，响应式）
- **编程语言**：Kotlin
- **棋盘引擎**：自研Canvas渲染 + SGF解析库
- **AI引擎**：KataGo Android移植（NNAPI加速）
- **数据存储**：Room数据库（题目、进度、统计）
- **题库格式**：SGF标准格式（兼容现有海量题库）
- **本地OCR**：ML Kit On-device Text Recognition

### 性能优化
- **懒加载**：题库按需加载，不一次性占用内存
- **图片压缩**：题目示意图使用WebP格式
- **增量更新**：新版题库通过差分包更新，减少下载量
- **后台预加载**：预测用户可能做的下一题，提前加载

## 题库来源（合法合规）
1. **公共领域古籍**：《玄玄棋经》、《发阳论》、《官子谱》等
2. **开源/共享题库**：GoGrinder项目包含8000+道经典题
3. **AI自动生成**：使用KataGo在标准棋盘上生成唯一正解的题目
4. **原创题目**：聘请业余高段/职业棋手创作专属题目
5. **用户UGC**：高级用户可自建题目包（SGF导入）

## 市场差异化
### 核心优势
1. **真正单机**：无需联网，地铁、飞机上随时做题，零流量消耗
2. **题量碾压**：15万+题，远超现有单机围棋App（GoGrinder仅8000题）
3. **AI赋能**：本地KataGo分析，每道题都有职业级解说
4. **拍照识题**：实体书题目一键数字化，打通线上线下学习
5. **完全免费**：无内购、无会员、无广告（可考虑捐赠版）
6. **开放格式**：SGF标准格式，用户可自行扩展题库
